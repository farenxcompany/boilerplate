# Pasta `types`

A pasta `types` é responsável por armazenar todas as definições de tipos e interfaces utilizadas no projeto. O objetivo principal é centralizar e padronizar os tipos para facilitar a manutenção e reutilização em toda a aplicação, garantindo maior segurança e consistência no uso do TypeScript.

## Estrutura da Pasta

Abaixo está uma estrutura exemplo de como a pasta `types` pode ser organizada:

```plaintext
src/
└── types/
    ├── api/
    │   ├── AuthTypes.ts
    │   ├── UserTypes.ts
    │   └── index.ts
    ├── components/
    │   ├── ButtonTypes.ts
    │   ├── ModalTypes.ts
    │   └── index.ts
    ├── global.d.ts
    ├── commonTypes.ts
    └── index.ts
```
